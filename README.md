# event-broker-api
RESTful API сервис для управления мероприятиями (событиями) с полным CRUD функционалом.

## 📋 Описание
API позволяет создавать, читать, обновлять и удалять мероприятия. Каждое событие содержит информацию о названии, времени проведения и описании.

## 🧰 Функциональность
| Метод | URL | Описание | Payload |
|-------|-------|-------|-------|
| **GET**         | /api/events |получение списка всех событий           |-|
| **GET/{id}**    | /api/events/{id:guid} |получение события по идентификатору |-|
| **POST**        | /api/events |создание нового события |json модель события (см ниже)|
| **PUT/{id}**    | /api/events/{id:guid} |полное обновление существующего события |json модель события (см ниже)|
| **DELETE/{id}** | /api/events/{id:guid} |удаление события |-|

### Параметры фильтрации метода GET
|параметр|тип|описание |примечание|
|-------|-------|-------|-------|
|Title|string|Фильтр по названию события|частичное совпадение, не зависит от регистра|
|From|string|Начало события|включительно (формат ISODateString)|
|To|string|Конец события|включительно(формат ISODateString)|
|ValidatePeriod|boolean|Включить проверку|что начало раньше завершения события,|
|Page|int|Номер страницы| по умолчанию 1|
|PageSize|int|Количество записей на странице|по умолчанию 10|
|OrderBy|string|Сортировка результатов| не реализовано|

пример запроса
**GET /api/events**?Title=конференция&From=2026-05-01T00:00:00Z&To=2026-05-31T23:59:59Z&ValidatePeriod=true&CheckPeriod=false&Page=1&PageSize=20&OrderBy=date%20desc

## 🛠 Технологии

- ASP.NET Core
- REST API
- Swagger/OpenAPI
- Entity Framework Core (предполагается)

## 📦 Модель данных

## Event (Событие)

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| Id | Guid | Да | Уникальный идентификатор события |
| Title | string | Да | Наименование мероприятия |
| StartAt | DateTime (ISO формат) | Да | Дата и время начала |
| EndAt | DateTime (ISO формат) | Да | Дата и время завершения |
| Description | string | Нет | Описание мероприятия |

### 🚀 Установка и запуск
1. откройте терминал
2. клонируйте проект git clone -b sprint_1 https://github.com/SerikovGennadiy/event-broker-api.git
3. перейдите в директорию cd event-broker-api (содержащую sln файл решения)
4. выполните dotnet run --project ./EventBrokerAPI/EventBrokerAPI.csproj
